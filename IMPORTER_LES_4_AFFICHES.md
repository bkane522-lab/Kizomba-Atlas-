# Ajouter les 4 affiches dans Supabase

1. Déployez ce ZIP complet sur le projet Kizomba Atlas.
2. Ouvrez Supabase > SQL Editor > New query.
3. Copiez tout le fichier `ajouter_4_dates_kizomba_atlas.sql`.
4. Collez puis appuyez sur `Run query`.
5. Actualisez `https://kizomba-atlas.vercel.app`.

Résultat : 3 fiches actives et 1 fiche Saïda conservée en brouillon, car le 14 juillet 2026 est déjà passé.
